# Projeto 2 — Itinerário e Meios de Transporte

O Projeto 2 consiste num sistema em Prolog para recomendação de itinerários e meios de transporte entre cidades. O sistema utiliza algoritmos de procura e otimização para encontrar caminhos de acordo com diferentes critérios.

O projeto encontra-se dividido em duas partes:

-Parte A — cálculo do caminho mais económico;
-Parte B — cálculo do caminho mais económico, mais rápido ou equilibrado entre custo e tempo.
