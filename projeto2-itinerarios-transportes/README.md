# Projeto 2 — Itinerário e Meios de Transporte

Este projeto foi desenvolvido no âmbito da unidade curricular de Técnicas de Inteligência Artificial.

O objetivo foi implementar um sistema em Prolog para recomendação de itinerários e meios de transporte entre cidades, recorrendo a algoritmos de procura e otimização.

## Estrutura

O projeto encontra-se dividido em duas partes:

```text
projeto2-itinerarios-transportes/
│
├── parte-a/
├── parte-b/
└── Relatorio_projeto2.pdf

#Parte A

A Parte A tem como objetivo encontrar o caminho mais económico entre uma cidade de origem e uma cidade de destino.

Foram utilizados os seguintes métodos:
Gerar e Testar;
Depth First;
Hill Climbing.

#Parte B

A Parte B acrescenta a dimensão do tempo de viagem, permitindo calcular:
o caminho mais económico;
o caminho mais rápido;
o caminho equilibrado entre custo e tempo.

#Relatório

O ficheiro Relatorio_projeto2.pdf contém a explicação detalhada do projeto, incluindo o enquadramento, objetivos, base de dados, métodos implementados, interface e conclusões.

Tecnologias utilizadas
Prolog;
SWI-Prolog.
