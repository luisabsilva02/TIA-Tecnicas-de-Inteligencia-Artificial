# Projetos em Prolog — Técnicas de Inteligência Artificial

Este repositório contém dois projetos académicos desenvolvidos no âmbito da unidade curricular de Técnicas de Inteligência Artificial, da Licenciatura em Engenharia e Gestão de Sistemas de Informação.

Os projetos foram implementados em Prolog, recorrendo ao SWI-Prolog, e abordam sistemas baseados em conhecimento, recomendação de meios de transporte, procura em grafos e algoritmos de otimização.

# Projeto 1 — Recomendação de Meio de Transporte

O Projeto 1 consiste num Sistema Baseado em Conhecimento desenvolvido em Prolog. O sistema interage com o utilizador através de perguntas e, com base nas respostas, recomenda o meio de transporte mais adequado.

O projeto encontra-se dividido em duas partes:
Parte A — aquisição manual de conhecimento;
Parte B — aquisição automática de conhecimento.

# Projeto 2 — Itinerário e Meios de Transporte

O Projeto 2 consiste num sistema em Prolog para recomendação de itinerários e meios de transporte entre cidades. O sistema utiliza algoritmos de procura e otimização para encontrar caminhos de acordo com diferentes critérios.

O projeto encontra-se dividido em duas partes:
Parte A — cálculo do caminho mais económico;
Parte B — cálculo do caminho mais económico, mais rápido ou equilibrado entre custo e tempo.

Tecnologias utilizadas
Prolog;
SWI-Prolog.
Autoria

Projetos académicos desenvolvidos no âmbito da unidade curricular de Técnicas de Inteligência Artificial.
