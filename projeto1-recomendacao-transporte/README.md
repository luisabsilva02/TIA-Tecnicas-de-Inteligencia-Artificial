# Projeto 1 — Recomendação de Meio de Transporte

O Projeto 1 consiste num Sistema Baseado em Conhecimento desenvolvido em Prolog. O sistema interage com o utilizador através de perguntas e, com base nas respostas, recomenda o meio de transporte mais adequado.

O projeto encontra-se dividido em duas partes:
-Parte A — aquisição manual de conhecimento;
-Parte B — aquisição automática de conhecimento
