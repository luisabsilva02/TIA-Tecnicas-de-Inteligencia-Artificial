# Projeto 1 — Recomendação de Meio de Transporte

Este projeto foi desenvolvido no âmbito da unidade curricular de Técnicas de Inteligência Artificial.

O objetivo foi implementar um Sistema Baseado em Conhecimento em Prolog para recomendar meios de transporte de acordo com as preferências e necessidades do utilizador.

## Estrutura

O projeto encontra-se dividido em duas partes:

```text
projeto1-recomendacao-transporte/
│
├── parte-a/
├── parte-b/
└── Relatorio_projeto1.pdf

#Parte A

A Parte A corresponde à aquisição manual de conhecimento. Nesta fase, foram criadas manualmente as regras da base de conhecimento, a base de dados, o sistema de inferência e a interface.

#Parte B

A Parte B corresponde à aquisição automática de conhecimento. Nesta fase, foram utilizadas técnicas de Data Mining, nomeadamente com recurso ao RapidMiner, para gerar regras que depois foram adaptadas para Prolog.

#Relatório

O ficheiro Relatorio_projeto1.pdf contém a explicação detalhada do projeto, incluindo o enquadramento, objetivos, implementação, funcionamento e conclusões.

Tecnologias utilizadas
Prolog;
SWI-Prolog;
RapidMiner.
